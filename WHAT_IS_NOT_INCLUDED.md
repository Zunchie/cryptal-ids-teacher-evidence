# What Is Not Included

This public teacher-view repository does not include:

- Full source code
- Private implementation files
- Raw run output
- Auth material
- Exchange/API configuration
- Hidden environment configuration
- Full AI chat export
- Private repository history
- Data files or private project internals
- Full workflow configuration
- Full tests

These exclusions are intentional. The goal is to provide school evidence without exposing the private project.
